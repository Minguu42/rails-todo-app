# rails-todo-app

## 概要

rails-todo-app は Ruby on Rails の学習目的で作成した ToDo アプリです.

ユーザ単位でプロジェクトを作成し, プロジェクトごとにタスクを管理することができます.

## 機能一覧

- ユーザー登録
- ログイン/ログアウト
- プロジェクトの作成/表示/更新/削除
- タスクの作成/表示/更新/削除
- プロジェクト、タスクのページネーション
